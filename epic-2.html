Epic 2: AI Feature Development & Backend Integration

Epic 2 implements EduGenie's core educational features, including Question & Answer, Concept Explanation, Quiz Generation, Content Summarization, and Personalized Recommendations. These features are powered by Google Gemini 1.5 Pro and LaMini-Flan-T5. It also develops and integrates FastAPI endpoints to enable seamless real-time communication between the frontend and the AI backend.

Key Objectives
  • Implement AI-powered Question & Answer functionality.
  • Generate detailed concept explanations for learners.
  • Create quizzes dynamically using AI models.
  • Summarize educational content effectively.
  • Provide personalized learning recommendations.
  • Develop RESTful APIs using FastAPI.
  • Connect frontend components with AI backend services.
  • Ensure real-time communication and fast response handling.

Technologies Used

AI Models: Google Gemini 1.5 Pro, LaMini-Flan-T5

Backend Framework: FastAPI

Purpose: AI-powered educational assistance with scalable API integration for real-time user interaction.
