EduGenie AI Modules

LaMini-Flan-T5 Model

EduGenie utilizes the LaMini-Flan-T5 model, a lightweight yet powerful generative AI designed to deliver educational content in a simplified and highly readable manner. The model is fine-tuned to generate concise, context-aware responses that break down complex topics into easily understandable language.

By emphasizing clarity and brevity, EduGenie enables beginners, school students, and self-learners to understand fundamental concepts without feeling overwhelmed. This makes LaMini-Flan-T5 an ideal study companion for foundational learning across multiple subjects.

Q&A Module

The Q&A Module is powered by Google Gemini 1.5 Pro, enabling EduGenie to answer both academic and general knowledge questions accurately. Its advanced contextual understanding provides reliable, intelligent, and detailed responses, making it an effective AI tutor across diverse educational topics.

Quiz Module

EduGenie automatically generates three multiple-choice questions (MCQs) from a given passage. Each question contains four carefully designed answer options generated using the Gemini 1.5 Pro model.

  • Analyzes the passage to understand its context.
  • Generates meaningful questions with four options.
  • Produces the output in structured JSON format.
  • Removes Markdown code blocks using the clean_json_block() function.
  • Parses the cleaned JSON into a Python list.
  • Provides detailed error messages if generation or parsing fails.

Summary Module

The Summary Module uses the Gemini model to transform lengthy paragraphs into concise, easy-to-understand summaries. It preserves essential information while eliminating unnecessary details, allowing learners to revise important concepts quickly without losing context or clarity.

Learning Path Module

The get_learning_recommendations() function leverages Google Gemini to generate personalized learning roadmaps for any topic.

  • Creates beginner-to-advanced learning paths.
  • Organizes topics according to difficulty levels.
  • Suggests useful resources such as videos, articles, and books.
  • Adapts recommendations based on the learner's knowledge level.
  • Safely handles API responses and exceptions.
  • Returns either structured recommendations or meaningful error messages.
