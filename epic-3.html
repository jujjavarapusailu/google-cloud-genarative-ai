Epic 3: User Interface Development

Epic 3 focuses on building EduGenie's user interface using HTML, CSS, and Jinja2 Templates. It creates a responsive, user-friendly interface featuring input areas and real-time result displays. The templates are integrated with the FastAPI backend, allowing users to interact seamlessly with AI-powered educational features.

Key Objectives
  • Develop a responsive and attractive user interface.
  • Create reusable Jinja2 templates for dynamic web pages.
  • Design user-friendly input forms and output sections.
  • Display AI-generated responses in real time.
  • Integrate frontend templates with FastAPI backend APIs.
  • Ensure smooth navigation and user experience across devices.
  • Connect users directly with EduGenie's AI-powered services.

Technologies Used

Frontend: HTML5, CSS3, Jinja2 Templates

Backend Integration: FastAPI

Purpose: To provide a responsive, dynamic, and interactive interface that enables users to access EduGenie's AI-powered educational features efficiently.
