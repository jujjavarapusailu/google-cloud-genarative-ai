Local Deployment of EduGenie

Overview

This activity focused on deploying and running the EduGenie Learning Assistant in a local development environment using FastAPI and Uvicorn. The deployment verified that the complete frontend-backend system was functioning correctly and could be accessed through a web browser for real-time interaction.

Application Execution

The application was started locally using the following command:

uvicorn main:app --reload

The --reload option automatically reloads the server whenever project files are modified, making development, debugging, and testing faster and more efficient.

Local Server Information

Parameter Details
Application Server FastAPI + Uvicorn
Execution Command uvicorn main:app --reload
Local Server URL http://127.0.0.1:8000
Access Method Web Browser

Terminal Output

After successful execution, the terminal displayed the following information:

  • Local server address (http://127.0.0.1:8000).
