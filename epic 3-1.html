Frontend Interface Development

Overview

This activity focused on developing the frontend interface of the EduGenie Learning Assistant using HTML, CSS, and Jinja2 Templates. The primary objective was to design a simple, responsive, and user-friendly interface that enables learners to interact easily with the AI-powered educational features.

The frontend was developed within the index.html file, while the visual appearance and responsiveness were managed through the style.css file.

Implemented Educational Features

Feature Description
Question Answering (Q&A) Allows users to ask educational questions and receive AI-generated answers.
Concept Explanation Provides simplified explanations for complex topics.
Quiz Generation Generates multiple-choice quizzes from user-provided content.
Text Summarization Converts lengthy educational text into concise summaries.
Learning Recommendations Creates personalized learning paths and study recommendations.

Frontend Components

  • User input fields for entering educational queries.
  • Task-specific placeholder text to guide users.
  • Submit buttons for sending requests to the backend.
  • Dedicated output containers for displaying AI-generated results.
  • Responsive page layout suitable for different screen sizes.

Frontend Styling (style.css)

  • Responsive web page design.
  • Modern button and textarea styling.
  • Well-organized result display containers.
  • Improved readability and user experience.
  • Clean and structured educational interface.

Outcome

This activity successfully established the user interaction layer of EduGenie.
