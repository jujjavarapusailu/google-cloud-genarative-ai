Frontend and Backend Integration

Overview

This activity focused on integrating the HTML frontend interface with the FastAPI backend to enable real-time communication between users and the AI-powered educational modules of the EduGenie Learning Assistant.

Form submissions from the frontend were connected to backend RESTful API endpoints using POST requests. This integration enabled seamless data exchange between the user interface and the AI-powered backend services.

Supported Educational Features

Feature Backend Integration

Key Functionalities

  • Captures user input through HTML forms.
  • Sends requests to FastAPI endpoints using POST requests.
  • Processes requests through AI-powered educational modules.
  • Returns structured JSON responses from the backend.
  • Displays generated results dynamically without manual page refresh.
  • Provides fast, smooth, and interactive user experience.
